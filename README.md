# Brew & Bloom Café Menu

A café menu page built with HTML and basic CSS.

This is one of my early projects as I work through learning frontend development. I built it to practice structuring a real page with HTML and styling it with CSS from scratch.

## What's in it
- Three menu sections: Coffee, Sweet, and Savoury
- Floated images per section
- Google Fonts (Playfair Display + Lato)
- Styled footer with contact info and links
- Background image with a centered ivory card layout

## Built with
- HTML5
- CSS3
